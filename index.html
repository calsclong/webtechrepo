Progressive Enhancement Writing Assignment

  Progressive Enhancement is a method of web development that makes the web more accessable by prioritizing content. It works by ordering web development into content first, presentation second, and client-side scripting third. By ordering web development this way, you are making it easier for your website to work on older browsers, because the website will work without the client-side scripting, and it will work without the presentation layer either. The content layer works on all browsers, and the further steps benefit modern browsers without adding necessary components. Progressive Enhancement is important for maintaining accessibility within the web by making sure websites are accessible from older browsers. Here is an article I found on the topic that goes into further detail: Building A Robust Frontend Using Progressive Enhancement.

Mars Rover Project Revision and Reflection

  When asking for how to improve my website, I was met with comments I had been thinking of myself. The main two were to make the website less bubbly and to make the website less flat. To make the website less bubbly, I changed the header and footer to span the entire page from left to right. This made the website feel less bubbly without incorporating sharp edges, because I instead removed there being edges on the header and footer. To make the website less flat, I replace the solid colored content backgrounds with a space-themed gradient. When I asked for review of the updated website, the css gradient was repeatedly noticed and complimented.
  I had many learning experiences while working on this assignment. One thing that frustrated me was that I could not figure out how to incorporate multiple CSS images, with above ones having transparency. I am not sure whether or not it is possible, but I attempted both links and multiple file types. Another thing I realized was that making a website more complicated or adding more css did not correlate to improving the website. In order to make the header and footer span the page, I had to remove CSS, not add it.

However, I learned far, far more through the course.
